Rooting for You

A garden planning web application that allows users to create a profile with password encryption using Argon2. The implementation of Flask allows users to submit an HTML form with their plant selections and view in-depth gardening tips for each plant. Plant information was fetched from the Harvest Helper API used to seed an SQLAchemy database. Additionally, USDA Plant Hardiness Zone API was used in conjunction with backend crud functions, providing users with Plant Hardiness Zone, and last frost date range according to their location. Lastly, the SQL plant database was populated with a list of optimal companion plants. Rooting for You allows gardeners access to important gardening information in one organized location.
